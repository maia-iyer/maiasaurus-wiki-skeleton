# Epistemic Stance

Frame claims as "X proposes..." / "according to Y..." — never as fact. Flag source type (proposal, draft, blog, RFC, spec, peer-reviewed). Surface contradictions with existing wiki content; do not overwrite.

Epistemic weight: peer-reviewed > spec > standard > doc > blog > proposal > other

## Mode-specific notes

- **Single-page**: epistemic weight varies by source type.
- **Repo**: repository documentation may be aspirational or out of date. Frame as "the README describes..." / "according to the project docs...". Note the browse date.
- **Site**: documentation may lag implementation. Frame as "the documentation states..." / "according to the <project> docs...". Note the crawl date.
